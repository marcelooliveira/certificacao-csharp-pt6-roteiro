# Como você deve completar o código relevante?

Você está desenvolvendo um aplicativo que inclui uma classe chamada `Cliente`.
O aplicativo produzirá a classe `Cliente` como um documento XML estruturado usando o
segmento de código a seguir:

```xml
<?xml version="1.0" encoding="utf-8"?>
<Contratante xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xmlns:xsd="http://www.w3.org/2001/XMLSchema"
ContratanteId="aaaaaaaa-1111-4444-bbbb-333333333333" xmlns="http://Contratante">
	<NomeCompleto></NomeCompleto>
	<DataNascimento>1990-07-08T00:00:00</DataNascimento>
</Contratante>
```

Você precisa garantir que a classe `Cliente` seja serializada corretamente para XML.
Como você deve completar as lacunas no código abaixo?

### código

```csharp
[LACUNA 1]
class Cliente
{
    [LACUNA 2]
    public Guid Id { get; set; }
    [LACUNA 3]
    public string Nome { get; set; }
    public DateTime DataNascimento { get; set; }
    [LACUNA 4]
    public int Email { get; set; }
}
```

### resposta

```csharp

class Cliente
{
    [XmlAttribute("ContratanteId")]
    public Guid Id { get; set; }
    [XmlElement("NomeCompleto")]
    public string Nome { get; set; }
    public DateTime DataNascimento { get; set; }
    [Xmllgnore]
    public int Email { get; set; }
}
```
