# Qual tipo de coleção você deve usar?

Você está desenvolvendo um aplicativo que inclui uma classe chamada `Pedido`. O aplicativo
armazenar uma coleção de objetos da classe `Pedido`.
A coleção deve atender aos seguintes requisitos:
* Armazenar internamente uma chave e um valor para cada item de coleção.
* Fornecer objetos aos iteradores em ordem crescente com base na chave.
* Assegure-se de que o item esteja acessível pelo índice baseado em zero ou pela chave.

Você precisa usar um tipo de coleção que atenda aos requisitos.
Qual tipo de coleção você deve usar?

A.
LinkedList

B.
Queue

C.
Array

D.
HashTable

E.
SortedList

### Resposta:

E. SortedList
