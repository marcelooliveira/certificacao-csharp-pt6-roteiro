# Qual código você deve usar para substituir a linha 05?

Você está desenvolvendo um aplicativo.
O aplicativo contém o seguinte segmento de código (os números de linha são incluídos
Apenas referência):

```csharp
01 ArrayList array1 = new ArrayList();
02 int var1 = 10;
03 int var2;
04 array1.Add(var1);
05 var2 = array1[0];
```

Quando você executa o código, você recebe a seguinte mensagem de erro: “Não é possível converter implicitamente
digite "objeto" para "int". Existe uma conversão explícita (você está perdendo um elenco?). ”
Você precisa garantir que o código possa ser compilado. Qual código você deve usar para substituir
linha 05?

A.
var2 = array1[0] is int;

B.
var2 = ( (List<int>)array1) [0];

C.
var2 = array1[0].Equals(typeof(int));

D.
var2 = (int) array1[0];

### resposta:
B.
var2 = ( (List<int>)array1)[0];

Faça uma lista de inteiros da matriz com = ((List<int>)array1) e selecione o primeiro item no
lista com [0].
