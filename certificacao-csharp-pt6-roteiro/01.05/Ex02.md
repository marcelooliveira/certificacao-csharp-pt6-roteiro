# Qual segmento de código você deve inserir na linha 20?

Você está desenvolvendo um aplicativo. O aplicativo converte um objeto `Localizacao` em uma string 
usando um método chamado `WriteObject`. O método `WriteObject()` aceita dois parâmetros, um
objeto `Localizacao` e um objeto `XmlObjectSerializer`.
O aplicativo inclui o seguinte código. (Os números de linha são incluídos apenas para referência.)

Você precisa serializar o objeto `Localizacao` como um objeto JSON.
Qual segmento de código você deve inserir na linha 20?

A.
New DataContractSerializer(typeof(Localizacao))

B.
New XmlSerializer(typeof(Localizacao))

C.
New NetDataContractSenalizer()

D.
New DataContractJsonSerializer(typeof(Localizacao))

### Resposta

D. New DataContractJsonSerializer(typeof(Localizacao))

Explicação:
A classe `DataContractJsonSerializer` serializa objetos para a notação de objeto JavaScript
(JSON) e desserializa dados JSON para objetos.
Use a classe `DataContractJsonSerializer` para serializar instâncias de um tipo em um JSON
documento e desserializar um documento JSON em uma instância de um tipo.
