# Como você deve completar o código relevante?

Um aplicativo serializa e desserializa código XML, que segue o seguinte formato:

```XML
<Nome xmlns="http://www.contoso.com/2012/06">
    <Sobrenome>Esponja</Sobrenome>
    <PrimeiroNome>Bob</PrimeiroNome>
</Nome>
```

O aplicativo lê os códigos XML usando um objeto `DataContractSerializer` que é
declarado pelo seguinte segmento de código C#:
`var ser = novo DataContractSerializer (typeof (Name));`

Você precisa garantir que o aplicativo preserve a ordem do elemento, conforme
o código XML de exemplo.
Como você deve completar o código relevante?

### Opções

```CSHARP
[DataContract(Namespace="http://www.contoso.com/2012/06")]
```
```CSHARP
[DataMember(Order=10)]
```
```CSHARP
[DataMember]
```
```CSHARP
[DataContract(Name="http://www.contoso.com/2012/06")]
```
```CSHARP
[DataMember(Name="http://www.contoso.com/2012/06", Order=10)]
```
```CSHARP
[DataContract]
```
```CSHARP
[DataMember(Name="http://www.contoso.com/2012/06")]
```

### Código

```
[LACUNA 1]
class Name
{
    [LACUNA 2]
    public string PrimeiroNome { get: set; }
    [LACUNA 3]
    public string Sobrenome { get; set: }
}
```

### Resposta

```
[DataContract(Namespace="http://www.contoso.com/2012/06")]
```
```
[DataMember(Order=10)]
```
```
[DataMember]
```
