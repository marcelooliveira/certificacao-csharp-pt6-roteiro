# Como você deve completar o código relevante?

Você está desenvolvendo um aplicativo que inclui uma classe chamada Customer.
O aplicativo produzirá a classe Customer como um documento XML estruturado usando o
Segmento de código a seguir:

```xml
<?xml version="1.0" encoding="utf-8"?>
<Prospect xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xmlns:xsd="http://www.w3.org/2001/XMLSchema"
ProspectId="aaaaaaaa-1111-4444-bbbb-333333333333" xmlns="http://prospect">
	<FullName></FullName>
	<DateOfBirth>1990-07-08T00:00:00</DateOfBirth>
</Prospect>
```

Você precisa garantir que a classe Customer seja serializada para XML.
Como você deve completar o código relevante? (Para responder, arraste o código apropriado
segmentos para os locais corretos na área de resposta. Cada segmento de código pode ser usado
uma vez, mais de uma vez ou nada. Pode ser necessário arrastar a barra de divisão entre painéis ou
desloque-se para ver o conteúdo.)
