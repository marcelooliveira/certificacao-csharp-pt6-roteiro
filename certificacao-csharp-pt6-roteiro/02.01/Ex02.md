Atribuindo Valores A Um Array
=============================

Manoel precisa declarar uma coleção de nomes de meses do ano. Para criá-la, ele decide declarar um array com tamanho suficiente para armazenar os nomes de todos os meses do ano (na ordem correta). Em seguida, Manoel tem que definir no array o valor para o mês de janeiro.

O que Manoel pode fazer? Escolha a melhor alternativa.




```
Array<string> meses = new Array[12];
meses[0] = "Janeiro";
```

> Ops! A sintaxe de declaração `Array<string>` é inválida em C#. Os arrays são declarados com o nome do tipo do elemento, seguido de colchetes.



```
string[] meses = new string[12];
meses[0] = "Janeiro";
```

> Isso aí!



```
string[] meses = new string[12];
meses[1] = "Janeiro";
```

> Ops! O índice 1 não é o mesmo que posição 1! Um array é sempre base zero, lembra? Então a primeira posição é índice zero.



```
string[] meses = new string[12];
meses["Janeiro"] = 1;
```

> Hein? `meses["Janeiro"]` não é sintaxe válida! Dentro dos colchetes deve vir o índice da posição do array que Manoel precisa preencher.
