# preencha cada declaração de acordo com as informações apresentadas no código.

Você tem o seguinte código:

```csharp
private static Dictionary<string, int> CriarSelecoes()
{
    Dictionary<string, int> dicionario = new Dictionary<string, int>()
    {
        {"Brasil", 5},
        {"Alemanha", 4},
        {"Itália", 4}
    };
    return dicionario;
}

private static bool EncontrarSelecao(string selecao)
{
    Dictionary<string, int> data = CriarSelecoes();

    if (data.ContainsKey(selecao))
    {
        return true;
    }
    else
    {
        return false;
    }
}
```

Para responder, preencha cada declaração de acordo com as informações apresentadas no código.

* Se o texto de busca é "França", o resultado será **[false|true|null]**
* Se o texto de busca é "5", o resultado será **[false|true|null]**
* Se o texto de busca é "Itália", o resultado será **[false|true|null]**

### Resposta:

* false
* false
* true
