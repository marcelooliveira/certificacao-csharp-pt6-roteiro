# Qual tipo de coleção você deve usar?

Você está desenvolvendo um aplicativo que inclui uma classe chamada **Order**. O aplicativo
armazenar uma coleção de objetos **Order**.
A coleção deve atender aos seguintes requisitos:
Use membros fortemente tipados.
Ordens de ordem de processo na ordem *first-in-first-out*.
Armazene valores para cada objeto **Order**.
Use índices baseados em zero.
Você precisa usar um tipo de coleção que atenda aos requisitos.
Qual tipo de coleção você deve usar?

A.
Queue<T>

B.
SortedList

C.
LinkedList<T>

D.
HashTable

E.
Array<T>


### Resposta:

A.
Queue<T>
